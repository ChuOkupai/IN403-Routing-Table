# IN403 - Projet Tables de routage - UVSQ

## Dépendances
Pour se compiler et s'exécuter, le projet fait en C a besoin de gcc et CSFML. L'installation sous Linux peut se faire avec la commande :
```shell
make install
```
### Compilation :
```shell
make rooting-table.out
```
### Exécution :
```shell
make
```
ou
```shell
make run
```
